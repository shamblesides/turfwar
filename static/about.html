Welcome to the IPv4 Games

I have a simple challenge for you:

Send requests to this web server from lots of different IP addresses.

How Scoring Works

The server keeps a record of what name was last sent to it from each IPv4 address.

If you hold more IPv4 addresses in a /8 address block than anyone else, that means you control that block.

Each block you control gives you 1 point.

Who made this?

Me!

How did you make this?

The frontend is plain HTML and JavaScript (just view the page's source!) and the backend is a couple hundred lines of Go with SQLite as the database.
